// How it works

Message → Pup → Code

Send a message in your chat. Get a persistent AI agent that writes code on your machine.

1
Send Message
Message your group on WhatsApp, Telegram, or Slack with a coding task.
2
Pup Spawns
bark-pack spawns a "pup" — a named agent running in a tmux session on your machine.
3
Live Updates
Your pup streams progress back to the chat. Reply to continue the conversation.

// Features

Your AI Army, Your Rules

Powerful features for managing your fleet of coding agents.

🔀
Multi-Backend
Claude Code, Cursor, Codex, Gemini — use any LLM backend. Switch mid-conversation or let fallback handle it.
💾
Persistent Sessions
Pups remember everything. Resume conversations hours or days later. Survives server restarts.
🎙️
Voice Messages
Send voice messages — transcribed locally with whisper.cpp. No cloud, fully private.
🔄
Auto-Recovery
Context window full? Rate limited? Crashed? bark-pack automatically recovers with context preserved.
📂
File Sharing
Pups can send files back to your chat — screenshots, exports, generated assets.
🎯
Cross-Backend Skills
Reusable prompt modules that work across all backends. Add "developer" or "architect" mode to any pup.

// Commands

Control Your Pack

Simple commands to manage your agents from any chat platform.

bark-pack — commands
$ /status # refresh pinned status
$ /stop Chase # stop a running pup
$ /reset Marshall # wipe memory, fresh start
$ /daily # standup from all pups
$ /backends # show available LLMs
$ /skill developer @Chase # add skill to pup
$ @Chase #opus fix the auth bug # route + model switch
