02
Features

Built for real
engineering work.

🤖

Multi-Backend LLM

Swap between Claude Code, Cursor, OpenAI Codex, and Gemini per pup. Just add #cursor or #gemini to your message.

♾️

Persistent Sessions

Each pup holds its full conversation history. Shelve it, reborn it later with intact memory. Sessions survive restarts.

🎙️

Voice Transcription

Send voice notes. They're transcribed locally via whisper.cpp (90+ languages) before being passed to the pup.

⚡

Auto-Recovery

Context window full? Rate-limited? The fallback engine retries, resets on same backend, or switches — with full context preserved.

📁

File Delivery

Pups drop files into their outbox. The server detects and delivers them to your chat automatically — any file type.

🐕

Pup Delegation

Pups spawn sub-agents with bark delegate. Add --branch for isolated work with an automatic PR.

03
Backends

Four engines.
One interface.

Tag your message to choose. Auto-failover moves between backends if something breaks — with context fully preserved.

🤖

Claude Code

Anthropic's CLI coding agent. Default backend, best for large-context engineering tasks and complex tool use.

#claude-code
🖱️

Cursor

AI-native code editor's CLI backend. Full session persistence and live streaming output.

#cursor
🧠

OpenAI Codex

OpenAI's CLI agent for code generation, analysis, and multi-step reasoning tasks.

#codex
✨

Google Gemini

Gemini CLI for multimodal input and broad general-knowledge coverage alongside coding.

#gemini
04
Commands

Run the pack
from any chat.

All commands work identically on WhatsApp, Telegram, and Slack. Use pack for bulk ops.

/status Refresh the pinned status message with all active pups.
/stop <name|pack> Stop a running pup or the whole pack.
/clear <name|pack> Shelve a pup. Memory preserved — can reborn later.
/reborn <name> Resurrect a shelved pup with full conversation history.
/reset <name|pack> Wipe pup memory — stays active on same backend.
/create Reply to a message to spawn a pup with that context.
/backends Show all available LLM backends and capabilities.
/daily One-line standup from every active pup.
/stats [name] Token usage and cost summary per pup or full pack.
/skills · /delete · /purge Manage skills, permanently delete pups, purge shelved.

06
Get started

Up and running
in minutes.

1

Clone & install

Grab the repo and run the prerequisites script — handles Homebrew, Node, tmux, and more.

git clone https://github.com/itayshmool/multi-bark-pack cd multi-bark-pack ./prerequisites.sh
2

Configure platforms

Copy .env.example to .env and add your Telegram token, WhatsApp group, or Slack credentials.

cp .env.example .env # Add TELEGRAM_TOKEN, WA_GROUP, # SLACK_BOT_TOKEN to .env
3

Install backends

Run the backend installer to set up and authenticate Claude Code, Cursor, Codex, or Gemini CLIs.

./install-backends.sh
4

Start the pack

Launch with auto-restart. Message your group — first pup spawns immediately.

yarn start # 🐾 First message → first pup
